# Making of Tic tac toe game (Probably didn't copied it from react.dev xD)

Tic tac toe is a class 2 player game

## Requirements

- There will be a 3x3 grid clickable squares.
- This games must be played with only 2 players. Each player having 1 turns.
- Turn-based gameplay, ensuring players alternate turns.
- Each player will must have a corresponding symbol of (X) and (O).
- A player wins by placing three of their symbols in a horizontal, vertical, or diagonal row.
- The game should display a winning message and highlight the winning combination.
